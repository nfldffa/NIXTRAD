## ðŸ“– Deskripsi
**NIXTRAD SYMMETRIC** adalah instrumen riset pasar kuantitatif yang mengintegrasikan data historis 10 tahun dengan model **Golden Ratio Projection**. Terminal ini memisahkan kebisingan pasar (market noise) dengan tren organik untuk menghasilkan proyeksi harga yang memiliki tingkat reliabilitas tinggi.



---

## âš¡ Fitur Unggulan

* **Dual-Floor Analytics**: Pemisahan panel antara pergerakan harga (Candlestick) dan kekuatan transaksi (Volume) untuk menghindari *visual clutter*.
* **Symmetric Bento Grid**: UI yang terinspirasi dari Bloomberg Terminal, memberikan data poin penting (Price, Target, ROI, Status) dalam satu pandangan simetris.
* **Skeptic Calibration Engine**: Algoritma cerdas yang menyesuaikan proyeksi berdasarkan RSI (Relative Strength Index) untuk mendeteksi kondisi *Overbought* atau *Oversold*.
* **Backtest Validation Mode**: Panel khusus untuk membandingkan proyeksi sistem dengan data riil 90 hari terakhir.
* **Multi-Asset Support**: Terintegrasi dengan Yahoo Finance API untuk menarik data IDX (Indonesia), Wall Street (US), dan Crypto secara *real-time*.
